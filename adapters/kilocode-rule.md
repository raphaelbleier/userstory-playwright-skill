# User Story Testing

This project has a user-story E2E testing skill.

When asked to generate user stories, build an E2E suite from the codebase, run a QA pass over the
site, or track bugs against user stories: read and follow the procedure in
`.agents/skills/user-story-testing/SKILL.md`, along with the reference files it points to in
`.agents/skills/user-story-testing/references/`.

Do not improvise a different workflow. The skill's scripts under
`.agents/skills/user-story-testing/scripts/` are the only supported way to read and write
`user-stories.xlsx`.
